# 上傳說明

## 方法一：Netlify Drop（最快速，30秒）

1. 打開瀏覽器訪問：https://app.netlify.com/drop
2. 把 `docs` 資料夾拖進去
3. 立即上線，獲得網址

## 方法二：Vercel（推薦）

1. 訪問 https://vercel.com/drop
2. 把 `docs` 資料夾拖進去
3. 立即上線

## 方法三：GitHub Pages

1. 創建 GitHub 倉庫
2. 上傳 docs 內容
3. Settings → Pages → 選擇 main branch

---

## 預覽本地網頁

直接用瀏覽器打開：`docs/index.html`

---

## 上傳後的訪問網址會像這樣：

- your-name.netlify.app
- your-project.vercel.app
- yourusername.github.io/repo-name
